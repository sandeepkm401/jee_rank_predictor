How the JEE-Main Rank Predictor Works

This page explains the machine learning approach used to estimate a student's All India Rank (AIR) based on the percentile they obtained in JEE-Main.

1. Objective of the Predictor

The goal of this system is to estimate the approximate All India Rank (AIR) for a given JEE-Main percentile. Since the official rank list depends on the performance of all candidates, the predictor uses historical data and machine learning to estimate the most probable rank range.

2. Dataset Used

The model is trained using a dataset containing approximately 5000 records. Each record represents a student and contains the following information:

  • Percentile obtained in JEE-Main
  • All India Rank (AIR) corresponding to that percentile

The dataset is stored in a CSV file:

jee_percentile_vs_air_5000_students.csv

The dataset represents realistic percentile-rank trends observed in previous JEE examinations.

3. Machine Learning Algorithm Used

This project uses the Linear Regression algorithm provided by the Scikit-Learn machine learning library.

Linear Regression attempts to learn a mathematical relationship between two variables:

  • Input Feature (X) → Percentile
  • Target Output (Y) → All India Rank

The algorithm tries to find the best possible line that fits the data points. This relationship can be expressed mathematically as:

Rank = m × Percentile + c

where:

  • m → slope of the regression line
  • c → intercept

These values are calculated automatically by the training algorithm.

4. Model Training Process

The following steps are used to train the machine learning model:

  1. Load the dataset using the Pandas library.
  2. Separate the dataset into:
    • Feature variable (X) → Percentile
    • Target variable (Y) → Rank
  3. Split the dataset into two parts:
    • Training Data (80%)
    • Testing Data (20%)
  4. Create a Linear Regression model.
  5. Train the model using the training dataset.
  6. The model learns the relationship between percentile and rank.

5. Prediction Process

Once the model is trained, it can predict the rank for any new percentile value.

The prediction process works as follows:

  1. User enters their JEE percentile in the predictor form.
  2. The input value is sent to the Django backend.
  3. The backend converts the percentile into numeric format.
  4. The trained Linear Regression model receives this value.
  5. The model calculates the predicted rank using the learned relationship.
  6. The predicted rank is returned to the webpage.
  7. The rank is displayed in the result field.

6. Example Prediction

If a student enters:

Percentile = 99.5

The trained model processes this input and may estimate a rank such as:

Predicted Rank ≈ 6200

Since real exam results vary slightly each year, the prediction is displayed with a small tolerance range.

7. Technologies Used

  • Python – core programming language
  • Django – backend web framework
  • Pandas – data processing and dataset handling
  • Scikit-Learn – machine learning model implementation
  • NumPy – numerical computation
  • Bootstrap – frontend design and styling

8. Important Note

The predicted rank is an approximation based on historical trends and machine learning estimation. Actual JEE ranks may vary depending on:

  • Exam difficulty
  • Number of candidates appearing for the exam
  • Score normalization process

Therefore, this tool should be used only for informational purposes.
